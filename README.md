Directions:
  1. Setup sender script on laptop/PC (for this one must use rewasd)
  2. Download and Install Arduino 1.8.13 (look it up off of somewhere other than Arduino's website)
  3. Download and Install Teensyduino 1.53 ([click this link](https://www.pjrc.com/teensy/td_153/))
  4. Get files from [NSGadget_Teensy](https://github.com/gdsports/NSGadget_Teensy)
  5. Setup/install script for teensy 4.1/4.0/3.6
  6. Start listener on teensy by plugging it in to a device
  7. Hook up ethernet from teensy to laptop/PC
  8. Run rewasd and then run sender script
  9. Have fun!
